# OGE Informatics Trainer (Node.js + React + SQLite)

Сайт для подготовки к ОГЭ по информатике:
- backend: `Express.js`
- frontend: `React (Vite)`
- база данных: `SQLite`

В базу предзагружены **10 вариантов** из раздела информатики на `inf-oge.sdamgia.ru`:
- 2507, 2508, 2509, 2510, 2511, 2512, 2513, 2514, 2515, 2516
- в каждом варианте загружены задачи **1-15**
- задачи **1-12** проверяются автоматически, задачи **13-15** отображаются и решаются с ручной проверкой
- картинки задач (там, где они есть в источнике) сохранены локально и отображаются на сайте

## Функции

- Тренажер по номеру задания (без таймера):
  - выбор номера задания
  - случайная задача выбранного номера
  - кнопка проверки и показ правильного ответа

- Полный вариант:
  - выбор варианта
  - решение всех задач с таймером **2 часа 30 минут**
  - для незалогиненного пользователя прогресс и таймер сохраняются в `localStorage`
  - после проверки показывается итоговый балл и разбор по каждому заданию

## Запуск одной командой (Docker)

```bash
docker compose up --build
```

После запуска сайт доступен на:
- `http://localhost:3000`

Для деплоя на VPS по домену см.:
- `DEPLOY.md`

## Локальный запуск без Docker

### 1) Backend

```bash
cd server
npm install
npm run dev
```

### 2) Frontend

```bash
cd client
npm install
npm run dev
```

Frontend в dev-режиме: `http://localhost:5173`.
API проксируется на `http://localhost:3000`.

## Структура

- `server/src/index.js` — API сервера
- `server/src/db.js` — SQLite, схема и сидирование
- `server/src/seed-data.json` — 10 вариантов и задания
- `client/src/pages/*` — страницы приложения
- `docker-compose.yml` + `Dockerfile` — production-упаковка
